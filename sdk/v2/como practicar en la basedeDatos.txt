Como practicar en la terminal


1. Alta (Insertar un nuevo usuario)
sqlite3 db.sqlite3 "INSERT INTO user (username, password, role) VALUES ('estudiante_diseno', 'clave2026', 'user');"

2. Baja (Eliminar un usuario)
Para borrar a alguien por su nombre de usuario:
sqlite3 db.sqlite3 "DELETE FROM user WHERE username = 'pepe_grillo';"

O más seguro, por su ID (ejemplo ID 23):

3. Modificación (Actualizar datos o permisos)
Para cambiar el rol de un usuario existente (por ejemplo, subir a jain a administrador):
sqlite3 db.sqlite3 "UPDATE user SET role = 'admin' WHERE username = 'jain';"


4. Consultas Rápidas (Querys de prueba)
Como la consigna pide experimentar con la base, puedes tirar consultas filtradas directamente:

Ver solo los administradores:
sqlite3 db.sqlite3 "SELECT * FROM user WHERE role = 'admin';"


Ver los últimos 5 usuarios agregados:
sqlite3 db.sqlite3 "SELECT * FROM user ORDER BY id DESC LIMIT 5;"
